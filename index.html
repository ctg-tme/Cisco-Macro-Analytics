About Cisco Macro Analyzer

The Cisco Macro Analyzer parses Macro Code to aid in documentation of your solution and/or troubleshoot existing solutions.

Once parsed, it will provide the following details of your Macro Code

  • Errors
    • Shows Common JS Syntax Errors
      • Performs same check as RoomOS Today
      • Adds additional check for generic use of the module object
      • Add additional check for use of module.name
      • Provides Search Regex and possible replacement suggestions
  • Warnings
    • Duplicate Subscriber Warnings
    • Duplicate xAPI Call Warnings
    • Use of Old xAPI Syntax
    • Use of Mixed xAPI Syntax
  • xAPI References
    • The Full Branch and Path
    • A Hyperlink to its documentation on the RoomOS Site
    • How it was used (Set, Get or Subscribed too)
    • Its recurrence in your script
